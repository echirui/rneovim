# rneovim Commands & Keybindings Reference

本プロジェクトでサポートする（または将来的にサポートする）Neovimの主要なキーバインドとコマンドのまとめです。

## モード切替
| キー | 説明 | 状態 |
|:---:|:---|:---:|
| `i` | 挿入モード (Insert mode) へ移行 | 実装済み |
| `:` | コマンドラインモード (Command-line mode) へ移行 | 実装済み |
| `/` | 検索モード (Search mode) へ移行 | 実装済み |
| `ESC` | ノーマルモード (Normal mode) へ戻る | 実装済み |
| `v` | ビジュアルモード (Visual mode) へ移行 | 未実装 |

## カーソル移動 (Motion)
| キー | 説明 | 状態 |
|:---:|:---|:---:|
| `h` | 左へ移動 | 実装済み |
| `j` | 下へ移動 | 実装済み |
| `k` | 上へ移動 | 実装済み |
| `l` | 右へ移動 | 実装済み |
| `w` | 次の単語の先頭へ移動 | 実装済み |
| `b` | 前の単語の先頭へ移動 | 実装済み |
| `e` | 単語の末尾へ移動 | 実装済み |
| `0` | 行頭へ移動 | 実装済み |
| `$` | 行末へ移動 | 実装済み |
| `gg` | ファイルの先頭へ移動 | 未実装 |
| `G` | ファイルの末尾へ移動 | 未実装 |
| `H`/`M`/`L` | 画面内の上端/中央/下端へ移動 | 未実装 |

## 編集 (Editing)
| キー | 説明 | 状態 |
|:---:|:---|:---:|
| `x` | カーソル下の1文字を削除 | 実装済み |
| `dd` | 現在の行を削除 | 暫定実装 (`d` 1回) |
| `y` | ヤンク（コピー） | 暫定実装 (1行) |
| `p` | ペースト（貼り付け） | 実装済み |
| `u` | 元に戻す (Undo) | 実装済み |
| `Ctrl-r` | やり直す (Redo) | 実装済み |
| `r` | 1文字置換 | 未実装 |
| `o` | 下に新しい行を挿入して挿入モードへ | 未実装 |
| `O` | 上に新しい行を挿入して挿入モードへ | 未実装 |
| `A` | 行末から挿入モードへ | 未実装 |

## コマンドライン (Ex Commands)
| コマンド | 説明 | 状態 |
|:---|:---|:---:|
| `:w [file]` | 保存 | 実装済み |
| `:q` | 終了 | 実装済み |
| `:wq` | 保存して終了 | 実装済み |
| `:e [file]` | ファイルを開く | 実装済み |
| `:echo [expr]` | 式を評価して表示 | 基礎実装済み |
| `:help` | ヘルプを表示 | 未実装 |

## 検索 (Search)
| キー | 説明 | 状態 |
|:---:|:---|:---:|
| `/pattern` | 前方に検索 | 実装済み |
| `?pattern` | 後方に検索 | 未実装 |
| `n` | 次の検索結果へ | 未実装 |
| `N` | 前の検索結果へ | 未実装 |

## その他
- `Ctrl-f`: 1ページ下にスクロール
- `Ctrl-b`: 1ページ上にスクロール
- `Ctrl-d`: 半ページ下にスクロール
- `Ctrl-u`: 半ページ上にスクロール
