# rneovim

NeovimをRustでスクラッチからポーティングするプロジェクトのプロトタイプです。
外部crateを一切使用せず、Rust標準ライブラリと最小限のC-FFIのみで構成されています。

## 特徴

- **純粋なRust実装**: 非同期イベントループ、バッファ管理、UIグリッドなどを標準ライブラリのみで実装。
- **Neovimアーキテクチャの継承**: RPCベースのAPIレイヤー、チャンク化されたテキスト保存 (MemLine) など、本家の思想を反映。
- **高機能エディタコア**:
  - **モードシステム**: Normal, Insert, CommandLine モード。
  - **効率的な描画**: 2次元グリッドによる差分更新エンジン。
  - **アンドゥ・リドゥ**: アクションベースの履歴管理。
  - **ファイルI/O**: ファイルの読み込みおよび `:w` による保存。

## クイックスタート

### ビルド
```bash
cargo build
```

### 実行
```bash
# 新規バッファで起動
./target/debug/rneovim

# ファイルを指定して起動
./target/debug/rneovim path/to/file.txt
```

### 基本操作
- **Normal Mode**:
  - `i`: 挿入モードへ移行
  - `:`: コマンドラインモードへ移行
  - `j` / `k`: カーソル上下移動
  - `u`: 元に戻す (Undo)
  - `Ctrl-r`: やり直す (Redo)
  - `q`: 終了
- **Insert Mode**:
  - `ESC`: ノーマルモードへ戻る
  - `Backspace`: 文字削除（行の結合に対応）
  - `Enter`: 改行（行の分割に対応）
- **Command Mode**:
  - `:w [path]`: 保存
  - `:q`: 終了
  - `:e path`: ファイルを開く
  - `ESC`: キャンセル
